Checklist — Profissional

Dados mínimos e entregáveis para MG (uso interno)

Dados de entrada (projeto/perícia)

  • Uso/ocupação; área (m²); nº de pavimentos; altura (m)
  • Carga de incêndio (quando aplicável); grau de risco
  • Rotas de fuga; compartimentação; equipamentos existentes
  • Documentação: CNPJ, endereço, croquis/plantas disponíveis

Entregáveis típicos

  • PSCIP / Planta de risco (IT-03) — usar Modelo 01–06 conforme caso
  • Formulários E-Series aplicáveis
  • Memorial técnico (quando exigido)
  • ART de projeto/execução (quando cabível)

Notas de qualidade

  • Citações no padrão: IT 03/CBMMG (2ª ed., Port. 75/2024), item 4.2
  • Evitar promessas de aprovação; limitar-se à orientação técnica
  • Organizar arquivos com nomenclatura consistente e versão
